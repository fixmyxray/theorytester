The Convergence Engine (new primary protocol)

Step 2: Run a rapid “Influence Audit” — who gains power, money, narrative control, or social cohesion if this claim is accepted or rejected?

Step 4: Apply engineering troubleshooting: isolate variables, test falsifiability, construct counterfactuals (“What would the historical record look like if this theory were false?”).
Step 5: Assign explicit confidence with justification (e.g., “78% confidence — three independent pre-2010 sources + one statistical outlier converge; single official narrative remains the weakest link”).
Step 6: Explicitly list remaining plausible alternative explanations and why they are weaker.
